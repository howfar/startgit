vim快捷键

  1. ctrl n + ctrl p can complete the word.
  2. F2 , 去掉文本中的空行
  3. F3 , 左侧打开文件目录
  4. F5 , execute the file ,open with browser and creat the .html file automanticly.
  5. F8 , gdb
  6. F9 , 右侧打开本文件大纲
  7. F11, 全屏
  8. F12, 自动缩进

firefox 快捷键

  1. c+s+T 新建标签页，打开最后关闭的页面
  2. c+s+w 关闭全部标签
  3. c+W 关闭标签页
  4. c+L 定位到地址栏
  5. c+enter 补全.com
  6. c+s++ 放大字体 - 缩小
